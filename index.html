Selectors and CSS Properties

We will start by checking some simple CSS concepts

The first set of elements contain selectors

Examples of Selectors

  • Universal Selector is * {...}
  • Type Selectors such as h1 {...}
  • Class selectors such as .list {...} or div.list {...}
  • ID selectors such as #top {...}

The second set contains CSS properties

Examples of Properties

  • color: green;
  • background-color: yellow;
  • font-size: 200%;
  • text-align: center;
